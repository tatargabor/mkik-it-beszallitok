## Context

Egyetlen `README.md` fájl, nincs architektúrális döntés, nincs kód. A tartalom a meglévő projekt struktúrából és az MKIK Vállalkozz Digitálisan oldalról származik.

## Goals / Non-Goals

**Goals:**
- Rövid, tömör, informatív README
- A weboldal link az elején, hogy azonnal elérhető legyen
- Módszertan leírása: adatgyűjtés folyamata
- Funkciók felsorolása

**Non-Goals:**
- Részletes API dokumentáció
- Telepítési / fejlesztői útmutató (ez egy egyszerű projekt)

## Decisions

### 1. Struktúra
1. Projekt cím + egy mondat leírás
2. Link a weboldalra (kiemelve)
3. Módszertan szekció
4. Funkciók szekció
5. Tech stack röviden
6. Futtatás (npm scripts)

### 2. Nyelv
Magyar - a projekt magyar nyelvű, a célközönség magyar.

## Risks / Trade-offs

Nincs lényeges kockázat.
